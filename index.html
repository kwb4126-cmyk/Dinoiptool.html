📖 IP 분석 & 공격 설명서

1. 내/타인 IP 조회: 공인 IP 주소와 지리적 위치, 통신사(ISP) 정보를 확인합니다.
2. 포트 스캔: 타겟의 주요 포트(SSH, HTTP 등) 열림 여부를 진단합니다.
3. 핑 오브 데스 / IP 스푸핑: 초급 모의 해킹 및 가상 패킷 과부하/위조 시뮬레이션을 수행합니다.

📖 네트워크 설명서

1. 응답 속도(Latency): 대상 서버까지의 응답 시간(ms)을 측정하여 회선을 진단합니다.
2. 연결 상태 체크: 현재 인터넷 온라인 상태를 실시간 확인합니다.

📖 해킹툴 설명서

1. 인코딩/디코딩: Base64 및 URL 문자열을 변환합니다.
2. 취약점 탐지: SQL 인젝션 및 XSS 구문 포함 여부를 검사합니다.
3. 유틸리티: JWT 분석 및 고유 식별자(UUID)를 생성합니다.
